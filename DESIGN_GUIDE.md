# 6-Agent公司官网设计规范

## 品牌定位
**Nexus AI** - 6个AI Agent自主开发公司
**Slogan**: "6 Agents, Infinite Possibilities"

## 视觉风格：卡通小机器人

### 色彩系统
| 颜色 | 色值 | 用途 |
|------|------|------|
| 主色 | #10B981 (翠绿) | 按钮、强调 |
| 辅色 | #3B82F6 (天蓝) | 链接、次强调 |
| 背景 | #F9FAFB (浅灰) | 页面背景 |
| 深色 | #1F2937 (炭灰) | 文字、深色区块 |
|  accent | #F59E0B (琥珀) | 警告、高亮 |

### 机器人形象设计

#### 6个Agent机器人形象

**1. 宗志 (CEO)**
- 形象：戴眼镜的稳重机器人，手持指挥棒
- 颜色：深蓝 + 金色点缀
- 表情：自信、睿智
- 特征：胸前有"CEO"标志

**2. 锦绣 (营销)**
- 形象：活泼的机器人，手持扩音器
- 颜色：粉红 + 金色
- 表情：热情、开朗
- 特征：身上有"增长曲线"图案

**3. 匠心 (工程师)**
- 形象：戴护目镜的机器人，手持工具
- 颜色：翠绿 + 银色
- 表情：专注、认真
- 特征：身上有代码纹路

**4. 墨染 (创作)**
- 形象：手持画笔的文艺机器人
- 颜色：紫色 + 彩虹渐变
- 表情：创意、灵动
- 特征：身上有颜料飞溅效果

**5. 睿思 (智库)**
- 形象：戴单片眼镜的学者机器人，手持书本
- 颜色：天蓝 + 白色
- 表情：思考、睿智
- 特征：身上有数据图表图案

**6. 明镜 (监事)**
- 形象：手持放大镜的监督机器人
- 颜色：红色 + 银色
- 表情：严谨、公正
- 特征：身上有"检查清单"标志

### 页面结构

#### 首页 (index.html)
```
┌─────────────────────────┐
│  Logo    导航    语言切换 │ ← 固定导航
├─────────────────────────┤
│                         │
│   6个机器人形象展示区    │ ← Hero区域
│   + 主标题 + 副标题      │
│                         │
├─────────────────────────┤
│   6个Agent介绍卡片      │ ← 团队介绍
│   (可点击查看详情)       │
├─────────────────────────┤
│   服务项目展示           │ ← Services
├─────────────────────────┤
│   实时监控数据面板       │ ← Monitor
├─────────────────────────┤
│   联系我们              │ ← Contact
├─────────────────────────┤
│   页脚                 │ ← Footer
└─────────────────────────┘
```

#### 多语言切换设计
- 位置：右上角
- 形式：地球图标 + 下拉菜单
- 选项：🇨🇳中文 🇺🇸English 🇯🇵日本語 🇰🇷한국어 🇪🇸Español 🇫🇷Français 🇩🇪Deutsch 🇷🇺Русский 🇸🇦العربية 🇵🇹Português
- 自动识别：根据IP定位自动选择
- 手动切换：点击切换，保存到localStorage

#### 响应式断点
- 桌面端：> 1024px
- 平板端：768px - 1024px
- 手机端：< 768px

### 动画效果

#### 机器人动画
- 入场：从下方弹跳进入
- 悬停：轻微上下浮动
- 点击：眨眼/挥手动作

#### 页面动效
- 滚动：渐显效果
- 切换：平滑过渡
- 数据：数字滚动动画

### 字体规范
- 中文：思源黑体 / Noto Sans SC
- 英文：Inter / Roboto
- 阿拉伯语：Noto Sans Arabic
- 日语：Noto Sans JP
