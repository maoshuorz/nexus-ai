# Nexus AI 官网设计规范 V2.0

## 设计风格：像素卡通（Pixel Art）

### 视觉风格
- **像素化设计**: 8-bit/16-bit风格，方块感强
- **卡通角色**: 6个Agent为像素风格小人
- **动画效果**: 
  - 角色走动（上下浮动+左右移动）
  - 气泡对话框弹出
  - 像素粒子效果
- **配色方案**: 复古游戏风格，高饱和度

## 动画规范

### 像素小人动画
```css
/* 走动动画 */
@keyframes pixelWalk {
  0%, 100% { transform: translateY(0) translateX(0); }
  25% { transform: translateY(-2px) translateX(5px); }
  50% { transform: translateY(0) translateX(10px); }
  75% { transform: translateY(-2px) translateX(15px); }
}

/* 气泡弹出 */
@keyframes bubblePop {
  0% { transform: scale(0); opacity: 0; }
  50% { transform: scale(1.1); opacity: 1; }
  100% { transform: scale(1); opacity: 1; }
}
```

### 实时数据更新
- **聊天内容**: 每5秒轮询API更新
- **项目状态**: 每30秒刷新
- **小人位置**: 随机走动，边界检测

## 技术实现

### 前端技术栈
- HTML5 + CSS3 (像素风格)
- JavaScript (动画 + API轮询)
- Canvas (可选，用于复杂像素动画)

### 响应式断点
- 桌面端: > 1024px (完整像素动画)
- 平板端: 768px - 1024px (简化动画)
- 手机端: < 768px (静态像素风格)

### 性能优化
- 像素图使用CSS绘制，减少图片加载
- API轮询使用节流技术
- 动画使用GPU加速
