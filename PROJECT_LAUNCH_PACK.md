### 项目1: Gmail自动接单系统

**目标**: 自动监控qingziyuezi@gmail.com邮箱，接收客户咨询并自动响应

**技术栈**:
- gog skill (已授权qingziyuezi@gmail.com)
- Python + 定时任务
- OpenClaw集成

**核心功能**:
```python
# 1. 监控新邮件
gog gmail messages search "in:inbox newer_than:1h" --account qingziyuezi@gmail.com --json

# 2. 识别咨询类型
- 项目咨询 → 转给COO分析
- 报价询问 → 转给CFO评估
- 技术支持 → 转给CTO处理
- 垃圾邮件 → 自动标记

# 3. 自动回复模板
- 收到咨询: "感谢您的咨询，我们的团队将在1小时内为您提供详细报价..."
- 报价发送: 包含项目评估、价格、交付周期
- 跟进提醒: 3天后无回复自动跟进
```

**Agent分工**:
- **COO**: 主负责，接收邮件并初步分析
- **CEO**: 审批报价
- **CFO**: 生成报价
- **CMO**: 记录客户来源

**预期收入**: 每单$2,000-$12,000

### 项目3: 客户跟进自动化

**目标**: COO工作流，自动跟进潜在客户，提高转化率

**工作流程**:
```
Day 0: 客户发送咨询邮件
  ↓ COO自动回复确认收到
Day 1: 发送详细报价
  ↓ 如果没有回复
Day 3: 发送跟进邮件1 (案例展示)
  ↓ 如果没有回复
Day 7: 发送跟进邮件2 (限时优惠)
  ↓ 如果没有回复
Day 14: 发送最后跟进 (保持联系)
  ↓ 标记为"需要人工介入"或"放弃"
```

**跟进模板**:
1. **确认收到**: "感谢您的咨询，我们已收到您的需求..."
2. **案例展示**: "我们最近完成了一个类似项目..."
3. **限时优惠**: "本周签约可享受9折优惠..."
4. **保持联系**: "如果您未来有需求，欢迎随时联系我们..."

**状态管理**:
- 新客户
- 已报价
- 跟进中
- 已成交
- 已放弃

**Agent分工**:
- **COO**: 主负责，执行跟进流程
- **CMO**: 提供营销话术
- **CFO**: 制定优惠策略
